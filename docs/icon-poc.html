Metis Trail — Icon Proof of Concept

Consistent SVG icon language for map markers, buttons, and inventory items.

Style rule

One rule: 2px stroke, rounded line caps, no fill, palette ink. Everything scales from this.

Map markers

One symbol per node type. High contrast at small size. Used on the trail strip and any map view.

Camp / Fire
flame pair + forest accent

Why SVG for icons

  • Scales to any size without pixelation
  • Single source of truth for color via stroke
  • Small file size; bundlable locally with the build

Location art approach

Each notable node gets a small scene header, not just a marker.
